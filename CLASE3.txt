API: Interface de Programación de Aplicaciones.

Mediante programación, empleando alguno de estos lenguajes:
Python, Javascript, C#, Java, RUST

1.) Abrir una terminal de command prompt desde el editor Visual Studio.


2.) Preparar el entorno del lenguaje con el cual se desarrollará
la integración con la API. En este caso se desarrollará en Python, 
por lo tanto debe ser instalado Python previamente.


3.) Preparar un entorno virtual.
Desde la línea de comandos, se debe ejecutar la instrucción:

Comando a realizar:
python -m venv .venv

Crear un entorno virtual, cuyo directorio se llamará \.venv


4.) Se debe activar el entorno virtual

>.venv\scripts\activate

(.venv)....>

¿Qué es el entorno virtual?

[Se instalan dependencias     ]
[Entorno Virtual              ]
[Interprete de lenguaje Python]
[Sistema Operativo            ]

¿Para qué sirve?
Sirve para separar ambientes, donde hay diferentes dependencias.

5.) Instalar la dependencia de Gemini IA, que se hace mediante el comando:

pip install -U google-genai     *************** IMPORTANTE *************

ver enlace:

¿Cómo verificar las dependencias?
pip list
